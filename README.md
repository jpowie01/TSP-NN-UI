# TSP NN UI
This repository is one of three parts that combines into one bigger project that resolves Travelling Salesman Problem using recurrent neural network.  

## Technologies
Used technologies:
- HTML5
- JS
- CSS

Planned:
- AngularJS

## Usage
Run API and open `index.html` in your browser.
